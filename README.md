### Usage

1. Clone the repository.
2. Install dependencies for the client:
    ```bash
    cd client
    npm install
    npm run dev
    ```
3. Install dependencies for the server:
    ```bash
    cd ../server
    npm install
    nodemon server.js
    ```
4. Open the app in your browser and start collaborating on documents in real-time.
